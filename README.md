# versioning
게임 영향 없는것은 a, b, c 등 끝에 알파벳 붙이기  
유닛 변경은 0.0.1 버전 올리기  
유닛 추가나 밸런스 크게 바뀌면 0.1 버전 올리기  
밸런스 완성도 높아졌으면 1.0.0 버전 올리기
